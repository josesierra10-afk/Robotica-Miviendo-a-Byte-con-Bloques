# Robotica-Miviendo-a-Byte-con-Bloques
Descubre cómo programar y dar vida a Byte, un robot educativo que se mueve con instrucciones visuales en bloques. Esta página ofrece actividades interactivas y recursos sencillos para aprender robótica y programación de manera divertida, ideal para estudiantes y docentes que inician en el mundo digital.
